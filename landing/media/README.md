# Медиа-файлы для лендинга

Положите сюда файлы:

| Файл | Описание | Формат |
|---|---|---|
| `demo.mp4` | Видео-демо управления записями (снятое на телефон) | MP4, вертикальное (9:16) |
| `screen-calendar.png` | Скриншот админ-панели — календарь | PNG |
| `screen-timeslots.png` | Скриншот админ-панели — слоты/чат | PNG |

Эти файлы отображаются на лендинге в секциях Hero (видео) и «Как это выглядит» (скриншоты).

При сборке (`npm run build:landing`) файлы копируются в `landing/dist/media/`.
