# Фича: Продающий лендинг

**Ветка:** `10-landing`
**Создана:** 2026-03-14
**Статус:** черновик

## Контекст

**Зачем:** Сейчас на `/` минимальная React-страница-заглушка с 3 шагами подключения. Нужен полноценный продающий лендинг, который:
- объясняет ценность сервиса владельцам бань
- снимает страхи («мы не ломаем ваш процесс»)
- даёт ссылку на Telegram-бота для подключения

**Что есть сейчас:** `LandingPage` — компонент внутри React SPA (`App.tsx`). Содержит заголовок, описание и 3 шага. Рендерится клиентским JS — плохо индексируется поисковиками. Текст не продающий, нет описания преимуществ.

**Ключевое ограничение:** лендинг должен быть **статическим HTML** (не SPA React), чтобы нормально индексировался Google и Яндекс. Будет раздаваться через nginx (спека `05-nginx-domain-https`). Ссылка на Telegram-бота берётся из ENV и подставляется при сборке или шаблонизации.

## User Stories

### US-1 — Владелец бани видит продающий лендинг (Приоритет: P1)

Владелец бани заходит на `domain.ru/` и видит красивую продающую страницу, которая объясняет зачем нужен сервис, снимает типичные возражения и даёт CTA — ссылку на Telegram-бота.

**Почему этот приоритет:** без лендинга нет конверсии — владелец не понимает зачем подключаться.

**Как проверить независимо:** открыть `/` в браузере, убедиться что контент отображается без JS (curl), мета-теги на месте, CTA ведёт на бота.

**Сценарии приёмки:**

1. **Дано** пользователь открывает `domain.ru/`, **Когда** страница загружается, **Тогда** он видит лендинг с описанием сервиса, преимуществами и кнопкой «Подключить»
2. **Дано** пользователь нажимает CTA-кнопку, **Когда** он кликает, **Тогда** открывается ссылка на Telegram-бота (`t.me/bot_username`)
3. **Дано** поисковый бот заходит на `/`, **Когда** он парсит HTML, **Тогда** весь контент доступен без выполнения JS (статический HTML)

---

### US-2 — Мобильная адаптивность (Приоритет: P1)

Лендинг корректно отображается на мобильных устройствах (320px+).

**Сценарии приёмки:**

1. **Дано** пользователь открывает лендинг на телефоне (ширина 375px), **Тогда** все секции читаемы, кнопки нажимаемы, горизонтального скролла нет

---

### US-3 — SEO мета-теги (Приоритет: P2)

Страница содержит правильные мета-теги для поисковиков и соцсетей.

**Сценарии приёмки:**

1. **Дано** поисковый бот парсит страницу, **Тогда** присутствуют: `title`, `meta description`, `og:title`, `og:description`, `og:image`

---

## Ключевые сущности

- **landing/index.html**: статический HTML-файл лендинга со встроенными стилями
- **BOT_URL**: ссылка на Telegram-бота, подставляется из env при сборке (или хардкодится при деплое)

## Контент лендинга

### Оффер (hero)

**Заголовок:** Онлайн-расписание бани — как тетрадь администратора, только не теряется и не путается

**Подзаголовок:** Все записи в одном месте. Ничего не теряется и не путается.

**CTA:** Подключить бесплатно → ссылка на бота

### Что это (позиционирование)

- Не агрегатор
- Не маркетплейс
- Не CRM
- Просто электронная тетрадь для записей

### Преимущества (на основе реального функционала)

1. **Расписание в Telegram** — задавайте рабочие часы и принимайте брони прямо в чате. Не нужно учить новые программы — всё работает в привычном мессенджере.
2. **Клиент сам выбирает время** — ваши клиенты видят свободные слоты на сайте и записываются без звонков. Меньше пропущенных вызовов, больше записей.
3. **Персональная страница** — каждая баня получает свою страницу с календарём. Отправьте ссылку клиентам — они сами разберутся.
4. **Ничего не ломаем** — можно продолжать принимать записи по телефону. Если не хотите — администратор сам всё вбивает. Мы просто убираем хаос.
5. **Естественное управление** — пишите боту как обычному собеседнику: «в пятницу бронь на 15:00 на 3 часа». Никаких форм и кнопок.
6. **Всё бесплатно** — сервис полностью бесплатный. Подключение за 2 минуты.

### Как это работает (шаги)

1. Напишите боту `/start` в Telegram
2. Назовите свою баню — бот создаст страницу
3. Задайте расписание — клиенты увидят свободные слоты

### Снятие возражений (FAQ / аккордеон)

- «А если я хочу продолжать принимать по телефону?» → Пожалуйста. Сервис не заменяет телефон, а дополняет.
- «Клиенты не будут пользоваться» → Им просто удобнее выбрать время, а не ждать ответа по телефону. Но если не хотите — администратор вбивает сам.
- «Это дорого?» → Бесплатно. Совсем.
- «Сложно подключить?» → 2 минуты в Telegram. Без установок, без регистраций на сайте.

## Edge Cases

- Что если ссылка на бота не задана в env? → CTA-кнопка показывает placeholder-текст «Скоро» и ведёт на `#`
- Должна ли страница бань `/:slug` оставаться React SPA? → Да, лендинг на `/` статический, страницы бань по-прежнему React

## Критерии приёмки

- [ ] Лендинг открывается по `/` и весь контент доступен без JS (статический HTML)
- [ ] Дизайн современный, красивый, адаптивный (320px+)
- [ ] CTA ведёт на Telegram-бота, ссылка берётся из env
- [ ] Присутствуют мета-теги: title, description, og:title, og:description
- [ ] Лендинг не ломает маршрутизацию `/:slug` → React SPA
- [ ] Стилистика согласована с существующим дизайном (Google Sans, синий акцент)
